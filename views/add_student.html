Intake Specifications

Absolute Requirements: All marked properties (*) must be evaluated with valid inputs prior to submission. Incomplete entries will cause backend API refusals.

Age Limits: Undergraduates matriculated at Global Tech must range from 16 to 80 years of age. Systems automatically flag outliers.

Security Guidelines: The Student PIN key allows candidate access to grade logs. Must contain 8+ characters, at least 1 uppercase letter, and 1 integer digit.

After successfully submitting, candidate information is merged into server active tables and lists. To confirm, navigate to the Directory console and filter by department.
